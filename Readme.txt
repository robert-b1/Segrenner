English

The name of the program "Segrenner" is a combination of the words 
"segregate" (the main idea was to segregate rubbish) and "scanner".

This program was created at the special request of my wife, who 
runs the Milvo.pl online store, which contains only ecological 
products (my wife is a bit "eco-freak" ). I had no idea for an 
application that I wanted to write as part of exercises for 
Android devices, so I consulted my wife and here it is.

Segrenner is a simple application that runs on android devices. 
After launching it, you can scan any QR code or barcode from, 
for example, food products. Then the program will take you to the
tab where you should describe the product by name, type of 
content, type of packaging. Then, you can save the entered data 
in a database created on a given device that we use. 
Appropriate active buttons of the application can move us to the 
list of already saved products and further allow you to edit or 
delete them. Segrenner is helpful in segregating waste from 
a given product, scanned and correctly described.

It is a program written mainly as part of my exercises in 
exploring the capabilities of the Android Studio environment 
and creating mobile applications. I hope that maybe it will be useful 
to someone in sorting rubbish.


Polski
Nazwa programu "Segrenner" powstała z połączenia wyrazów 
"segregować"(głównym założeniem była segregacja śmieci) 
i "scanner".

Program ten powstał na specjalne życzenie mojej żony, 
która prowadzi sklep internetowy Milvo.pl, na którym znajdują 
się same ekologiczne produkty (żona to trochę taki "eko-świr"). 
Nie miałem pomysłu na aplikację, którą chciałem napisać w ramach 
ćwiczeń na urządzenia z androidem, więc poradziłem się żony 
i oto jest.

Segrenner jest prostą aplikacją działającą na urządzeniach z 
androidem. Po jej uruchomieniu można zeskanować dowolny kod QR 
lub kod kreskowy z na przykład produktów spożywczych. Następnie 
program przeniesie nas do zakładki, gdzie należy opisać dany 
produkt po nazwie, oraz pod względem rodzaju zawartości, 
rodzaju opakowania. Następnie można zapisać wprowadzone dane 
w bazię danych utworzonej na danym urządzeniu, którym się 
posługujemy. Odpowiednie aktywne guziki aplikacji mogą przenieść 
nas do listy zapisanych już produktów i dalej pozwolić na ich 
edytowanie lub usuwanie. Segrenner jest pomocny w segregowaniu 
odpadów z danego produktu zeskanowanego i poprawnie opisanego. 

Jest to program napisany głównie w ramach moich ćwiczeń w 
poznawaniu możliwości środowiska Android Studio i tworzeniu 
aplikacji mobilnych. Mam nadzieję że może się kiedy komuś przyda 
w segregowaniu śmieci.
